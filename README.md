# Home

Here you will find all documentations of Dusa scripts

### What will you find here? <a href="#what-will-you-find-here" id="what-will-you-find-here"></a>

The information you will find here will be categorized under two main categories.

### **1 - General Documentation:** <a href="#id-1-general-documentation" id="id-1-general-documentation"></a>

This section consists of configurations, demonstration and description of the script in general.

### **2 - Developer Documentation:** <a href="#id-2-developer-documentation" id="id-2-developer-documentation"></a>

This section is intended for experienced developers who are familiar with server-side development. It explores more complex aspects of the system and may not be of immediate interest to non-technical users. Sharing this section with your server developers will ensure a smoother and more productive experience. This chapter consists of 3 parts.

#### **Functions:**

This is the section where the functions created for you to use externally are explained in general terms and what they do.

#### **Events:**

This is the section where the general usage of the events created for you to use externally and what they do are explained.&#x20;

#### **Config:**

This section contains the default config settings. If you make any changes and cannot resolve the error, you can return to your old settings here.

### Meaning of documentation codes <a href="#meaning-of-documentation-codes" id="meaning-of-documentation-codes"></a>

The purpose of the codes generated from random combinations that you encounter while reading the documentation is to enable you to quickly find solutions to problems or questions by searching for the code if the solution exists in the documentation. You can enter this code into the search bar to access solutions.
