# Tournaments

Compete against other hunters in timed events to earn exclusive rewards and claim the top spot on the leaderboard.

## Overview

Tournaments are competitive hunting events where players compete for the highest score within a time limit.

**Key Features:**
- Scheduled events (weekends, configurable)
- Real-time leaderboard
- Point-based scoring system
- Exclusive prizes for top players
- Time-limited (1-3 hours typically)

- First player to kill rare animal: +100 points

## Tournament Types

### Standard Tournament

Classic highest-score competition.

**Format:**
- All animals count
- All zones available
- 2 hour duration

### Themed Tournaments

**Predator Hunt:**
- Only predator kills count
- Bonus points for dangerous animals
- Risk vs reward gameplay

**Perfect Hunt:**
- Only perfect quality kills count
- High skill requirement
- Lower scores, higher rewards

**Speed Hunt:**
- 1 hour duration
- Double speed bonuses
- Fast-paced action

**Team Tournament:**
- 2-4 player teams
- Shared score pool
- Split rewards among team

## Troubleshooting

**Not Receiving Points**
- Ensure tournament is active
- Check if in tournament zone
- Verify animal harvested properly
- Relog if issue persists

**Can't See Leaderboard**
- Check keybind settings
- Ensure tournament started
- Try alternate access (menu)

**Disconnected During Tournament**
- Rejoin immediately
- Points should be saved
- Contact admin if score lost
