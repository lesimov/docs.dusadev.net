# Quest System

Complete daily and weekly quests to earn bonus rewards, XP, and unlock special content.

## Quest Types

### Daily Quests

Reset every day at 6 AM server time (configurable).

**Characteristics:**
- 3 active quests maximum
- Easy to medium difficulty
- Moderate rewards
- Quick completion (15-30 minutes)

**Example Quests:**
- Hunt 5 Deer
- Obtain 3 Perfect Quality Pelts
- Cook 10 Meals at Campfire
- Use 5 Traps Successfully
- Harvest 15 Animals

### Weekly Quests

Reset every Monday (configurable).

**Characteristics:**
- 1 active quest
- High difficulty
- Premium rewards
- Extended completion time

**Example Quests:**
- Hunt 50 Total Animals
- Obtain 20 Perfect Quality Pelts
- Reach Hunting Level 10
- Complete 15 Daily Quests
- Earn $10,000 from Hunting

### Special Quests

Event-based or one-time quests from NPCs.

**Sources:**
- Quest giver NPCs in hunting shops
- Seasonal events
- Achievement unlocks
- Admin-triggered events

## Quest Interface

### Accessing Quests

**Method 1: Hunting Menu**
- Open hunting menu (default: F7)
- Navigate to "Quests" tab
- View all active and available quests

**Method 2: Quest NPCs**
- Visit hunting shop
- Talk to quest giver NPC
- Browse and accept quests

### Quest Information Display

Each quest shows:
- Title and description
- Objectives with progress bars
- Time remaining (for daily/weekly)
- Rewards preview
- Difficulty indicator

## Quest Objectives

### Objective Types

**Kill Objectives**
```yaml
Hunt X amount of [Animal Type]
Example: "Hunt 5 Deer" (0/5)
```

**Quality Objectives**
```yaml
Obtain X [Quality] pelts
Example: "Get 10 Perfect Pelts" (3/10)
```

**Activity Objectives**
```yaml
Perform X [Action]
Example: "Cook 15 Meals" (7/15)
```

**Collection Objectives**
```yaml
Collect X [Item]
Example: "Gather 20 Deer Meat" (12/20)
```

**Location Objectives**
```yaml
Hunt in [Zone]
Example: "Hunt 10 animals in Mount Chiliad" (4/10)
```

## Rewards

### Reward Types

| Reward Type | Description |
|-------------|-------------|
| **Money** | Direct cash payment |
| **XP** | Hunting experience points |
| **Items** | Quest-exclusive items or supplies |
| **Unlocks** | Access to new zones or features |

### Reward Tiers

**Daily Quest Rewards:**
- Money: $500-1,500
- XP: 100-500
- Items: Common supplies (bait, traps)

**Weekly Quest Rewards:**
- Money: $3,000-10,000
- XP: 1,000-5,000
- Items: Rare items (legendary bait, exclusive weapons)

**Special Quest Rewards:**
- Unique cosmetics
- Title/badge unlocks
- Permanent buffs
- Exclusive access

## Completing Quests

1. **Accept Quest** from menu or NPC
2. **Track Progress** in hunting menu
3. **Complete Objectives** through normal gameplay
4. **Return to NPC** or auto-complete
5. **Claim Rewards** from quest menu

### Auto-Completion

Some quests complete automatically:
- No NPC return required
- Rewards sent directly to inventory
- Notification on completion
- Next quest unlocked

### Manual Turn-In

Other quests require NPC interaction:
- Return to quest giver
- Confirm completion
- Receive bonus dialogue
- Possible follow-up quest

## Quest Strategies

### Daily Quest Optimization

**Morning Routine:**
1. Check daily quests at 6 AM reset
2. Accept all 3 quests
3. Plan hunting route for efficiency
4. Complete easiest quest first
5. Chain quest objectives when possible

**Example Synergy:**
- Quest 1: Hunt 5 Deer
- Quest 2: Get 3 Perfect Pelts
- Quest 3: Cook 10 Meals

**Route**: Hunt deer for Quest 1+2, cook meat for Quest 3

### Weekly Quest Planning

**Long-Term Approach:**
- Check weekly quest on Monday
- Break into daily sub-goals
- Track progress throughout week
- Final push on Sunday if needed

**Example:**
- Weekly: Hunt 50 Animals
- Daily Goal: 7-8 animals per day
- Finish early for other activities

### Quest Stacking

Complete multiple quests simultaneously:

```
Active Quests:
- Hunt 10 Animals (any)
- Hunt 5 Deer (specific)
- Get 5 Perfect Pelts (quality)
- Use 10 Traps (activity)

Strategy:
1. Place 10 traps in deer zone
2. Hunt deer with headshots
3. All 4 quests progress together
```

### Quest Notifications

You'll receive notifications for:
- New quest available

## Common Quest Issues

**Quest Not Progressing**
- Ensure you're in correct zone
- Check if using correct animal/item
- Verify quest is accepted
- Try relogging if stuck

**Can't Accept Quest**
- Quest log may be full
- Level requirement not met
- Previous quest not completed
- Server-side cooldown active

**Rewards Not Received**
- Check inventory space
- Verify quest completion
- Claim from quest menu
- Contact admin if persists
