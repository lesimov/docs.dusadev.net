# Hunting

## Overview

Welcome to the comprehensive Hunting script for FiveM! This advanced hunting system brings an immersive wildlife experience to your server, featuring realistic hunting mechanics, camping, cooking, trapping, and a complete progression system.

## Key Features

✅ **Realistic Hunting System**
- Multiple animal types with unique behaviors
- Weapon-based damage system
- Animal quality and rating system
- Illegal hunting detection

✅ **Camping & Cooking**
- Deployable campsites with campfires
- Cooking system with recipes
- Process raw meat into consumables
- Tent deployment for immersion

✅ **Trapping & Baiting**
- Place traps to catch animals
- Use bait to attract wildlife
- Collect trapped animals
- Strategic hunting mechanics

✅ **Quest System**
- Daily and weekly quests
- NPC quest givers
- Reward system with items and money
- Track quest progress

✅ **Tournament System**
- Competitive hunting events
- Leaderboards and rankings
- Special rewards for winners
- Timed hunting challenges

✅ **Progression System**
- Level up by hunting
- Unlock new areas and animals
- Skill-based rewards
- Experience tracking

✅ **Framework Compatibility**
- ESX Legacy
- QBCore
- QBOX
- Unified bridge system (dusa_bridge)
