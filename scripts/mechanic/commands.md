# Commands

-----

| Command           | Description                                                | Required Perm |
| ----------------- | ---------------------------------------------------------- | ------------- |
| `/mechaniceditor` | Opens Mechanic Admin Editor                                | Admin         |
| `/tablet`         | Opens tablet if command is enabled in `dusa_tablet` config | Everyone      |
| `/redline`        | Opens Redline app if `dusa_tablet` isn't available         | Everyone      |
