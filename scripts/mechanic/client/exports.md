## Client Exports

### Show / Hide Text UI

**Showing Text UI**

```lua
exports.dusa_bridge:ShowTextUI(locale('zones.tuning.prompt'), {
    position = 'right-center',
    key = 'E',
})
```

**Hiding Text UI**

```lua
exports.dusa_bridge:HideTextUI()
```
