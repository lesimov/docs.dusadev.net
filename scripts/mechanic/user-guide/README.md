## Quick References

### Common Tasks

**Opening the Tablet**
1. Use the `mechanic_tablet` item from your inventory
2. The tablet interface opens automatically
3. Navigate to the Mechanic app

**Creating a Work Order**
1. Tablet → Mechanic app → Dashboard
2. Click "New Work Order"
3. Fill in vehicle and customer details
4. Add services and parts
5. Set pricing and create

**Tuning a Vehicle**
1. Get in vehicle as driver
2. Drive to tuning zone
3. Tablet → Mechanic app → Tuning
4. Select category and modifications
5. Preview and apply

**Checking Parts Stock**
1. Tablet → Mechanic app → Parts
2. View current inventory levels
3. Order low-stock items

**Managing Shop (Boss Only)**
1. Tablet → Mechanic app → Management
2. Access employees, finances, or settings
3. Make changes as needed

### Tips for Success

**For Mechanics:**
- Always update work order status as you progress
- Document all work in order notes
- Keep track of parts used
- Communicate with team members
- Test drive after major repairs or tuning

**For Shop Owners:**
- Monitor inventory levels regularly
- Keep popular parts well-stocked
- Set competitive but profitable pricing
- Review employee performance
- Maintain adequate shop balance for operations

## Keyboard Shortcuts

The tablet supports keyboard navigation:
- **ESC** - Close tablet
- **Arrow Keys** - Navigate (if enabled)
- **Enter** - Confirm/Select
- **Backspace** - Go back

**Note**: Keyboard shortcuts depend on tablet configuration.

## Best Practices

### Work Order Management
1. Create detailed descriptions
2. Add photos if available
3. List all required parts upfront
4. Update status promptly
5. Add completion notes

### Vehicle Tuning
1. Discuss options with customer first
2. Explain costs before proceeding
3. Test drive after modifications
4. Save popular configurations as presets
5. Document all changes

### Parts Inventory
1. Set minimum stock levels
2. Order before running out
3. Track fast-moving items
4. Review usage patterns monthly
5. Coordinate with team on orders

### Customer Service
1. Keep customers informed
2. Provide accurate time estimates
3. Explain charges clearly
4. Offer recommendations
5. Follow up after service
