# Mechanic

## Overview

The DUSA Mechanic system is a comprehensive vehicle service and customization solution designed for modern FiveM roleplay servers. Built on a tablet-based interface platform, it provides mechanics with professional tools to manage customer repairs, track work orders, and perform advanced vehicle modifications.

The system features a complete work order management system, an advanced vehicle tuning platform with performance and cosmetic modifications, parts inventory tracking, and multi-shop support. Whether you're running a single garage or managing multiple locations across your city, DUSA Mechanic delivers a seamless experience for both mechanics and customers.

At its core, DUSA Mechanic is framework-agnostic, working seamlessly with ESX Legacy, QBCore, and QBox through an intelligent abstraction layer. This ensures compatibility with your existing server infrastructure while maintaining a consistent, high-quality experience.

## Key Features

- **Tablet-Based Interface** - iPad-style app platform providing intuitive access to all mechanic functions
- **Complete Work Order System** - Track customer repairs from initial request to completion with detailed history
- **Advanced Vehicle Tuning** - Performance upgrades, cosmetic modifications, stance adjustments, and engine swaps
- **Parts Inventory Management** - Track parts stock, manage ordering, and automate inventory deduction
- **Employee Management** - Hire employees, assign roles, set permissions, and manage multiple mechanics
- **Multi-Shop Support** - Run multiple mechanic shops with separate employees, inventories, and finances
- **Framework-Agnostic** - Works with ESX Legacy, QBCore, and QBox without code modifications
- **Real-Time Synchronization** - All work orders and shop data sync instantly across all employees
- **Financial Management** - Shop account system with transaction history and balance tracking
- **Zone-Based Operations** - Define service zones for repairs and tuning areas

## What's Included

The DUSA Mechanic ecosystem consists of three integrated components:

- **dusa_bridge** - Framework abstraction layer providing unified API for all framework operations
- **dusa_tablet** - iPad-style tablet platform hosting the mechanic app and future applications
- **dusa_mechanicv2** - Core mechanic system with work orders, tuning, and parts management
