#### **Install** <a href="#install" id="install"></a>

Login to your [keymaster](https://keymaster.fivem.net/asset-grants/) account, check Granted Assets section and download script you bought from us

### **Script Start Order For ESX** <a href="#integrations-compatibilities" id="integrations-compatibilities"></a>

To correctly start the mdt script in your `server.cfg`, ensure to start the scripts in the following order. Failure to do so may result in various error codes

```
ensure es_extended
ensure ox_lib
ensure dusa_mdt
```

### &#x20;**Script Start Order For QBCore** <a href="#integrations-compatibilities" id="integrations-compatibilities"></a>

To correctly start the mdt script in your `server.cfg`, ensure to start the scripts in the following order. Failure to do so may result in various error codes

```
ensure qb-core
ensure ox_lib
ensure dusa_mdt
```
