**Install**

Login to your [keymaster](https://keymaster.fivem.net/asset-grants/) account, check Granted Assets section and download script you bought from us

**Script Start Order For ESX**

```
ensure es_extended
ensure dusa_multicharacter
ensure dusa_spawnselector
```

**Script Start Order For QBCore**

```
ensure qb-core
ensure dusa_multicharacter
ensure dusa_spawnselector
```

**Run SQL file**

You will find insert.sql file inside your dusa\_multicharacter script, open it on your database and press run for inserting
