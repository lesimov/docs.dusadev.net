# Installation

## Step-by-step

{% stepper %}
{% step %}
### Download dusa\_hud from your keymaster portal

You can find [keymaster portal](https://portal.cfx.re/assets/granted-assets) link from here
{% endstep %}

{% step %}
### Unzip dusa\_hud to your server resources folder


{% endstep %}

{% step %}
### Run SQL file

Open run.sql and run it to your database
{% endstep %}

{% step %}
## Add sounds to your sound resource

Sounds are included at /sounds folder

1. For interact-sound, head into interact-sound/client/html/sounds/ and move all sounds to there
2. For xsound, head into xsound/html/sounds/ and move all sounds to there
{% endstep %}
