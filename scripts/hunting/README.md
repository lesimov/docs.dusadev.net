# Hunting System

Welcome to the **Dusa Hunting System** developer documentation. This guide is designed for developers who want to integrate, customize, or extend the hunting system in their FiveM server.

## Overview

Dusa Hunting is a comprehensive hunting system for FiveM servers that provides a realistic hunting experience with features including animal AI, trapping, butchering, cooking, leveling system, quests, tournaments, and more.

## Quick Links

* **Framework Support**: QBCore, ESX, QBox (via dusa\_bridge)
* **Dependencies**:
  * ox\_lib (required)
  * dusa\_bridge (required)
  * MugShotBase64 (optional, for ID cards)

## Key Features for Developers

* **Quest System** - Create custom hunting quests with rewards
* **Level System** - Player progression with XP and titles
* **Tournament System** - Competitive hunting events
* **Quality System** - Item quality based on shot accuracy
* **Animal Spawning** - Dynamic animal spawning in zones
* **Butchering System** - Convert carcasses into meat and hide
* **Cooking System** - Cook raw meat with campfires/grills
* **Pet System** - Hunting companion animals <mark style="color:$info;">(Not Available Yet)</mark>
* **Trap System** - Place traps to catch animals
